# pixel-rgb-analyzer
Web app that extracts and visualizes RGB color data from uploaded images using JavaScript, SQL, and Chart.js

## Features
- Upload images and extract RGB values per pixel using the Canvas API  
- Dynamic color plotting with Chart.js  
- Data storage and timestamp grouping using PHP + SQL  

## Tech Stack
- JavaScript, HTML/CSS  
- PHP + MySQL  
- Chart.js for visualization  

## How to Run
1. Clone the repository  
2. Set up a local server (e.g. XAMPP or MAMP)  
3. Import the `shape_data` table schema into MySQL  
4. Open `index.html` in your browser
